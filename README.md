# 📚 Academic Data Breach Management System

The **Academic Data Breach Management System** is a full-stack web application designed to securely manage and monitor academic data in educational institutions.
The platform provides students with a dashboard to track academic performance and manage leave requests while ensuring secure access to sensitive academic information.

---

## 🚀 Features

* 🔐 **Secure Authentication** – Student login and access control
* 📊 **Dashboard Analytics** – CGPA trends and academic performance visualization
* 📝 **Leave Request System** – Apply for and track leave requests
* 📈 **Academic Monitoring** – Semester-wise progress tracking
* ⚡ **Responsive UI** – Modern interface built with React and Tailwind CSS

---

## ⚙️ Installation

### 1️⃣ Clone Repository

```
git clone https://github.com/your-username/academic-data-breach-management-system.git
```

### 2️⃣ Backend Setup

```
cd backend
mvn spring-boot:run
```

### 3️⃣ Frontend Setup

```
cd frontend
npm install
npm start
```
